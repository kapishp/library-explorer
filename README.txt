Library Explorer
================

A tool for visualizing and analyzing spectral libraries.

REQUIREMENTS
------------
- Python 3 must be installed on your computer
- Download from python.org if you don't have it

HOW TO RUN
----------
Mac:
1. Double click "run_mac.command"
2. If you get a security warning, go to System Settings -> Privacy & Security and click "Open Anyway"
3. Wait about 15 seconds for the app to start
4. The app will open automatically in your browser

Windows:
1. Double click "run_windows.bat"
2. Wait about 15 seconds for the app to start
3. The app will open automatically in your browser

NOTE: The first time you run the app it will take a few extra minutes to install dependencies. After that it will start up quickly.

STOPPING THE APP
----------------
Mac: Close the terminal window that opened
Windows: Close the command prompt window that opened
